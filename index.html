Introduction

These maps estimate the magnitude of mosquito reports in Collier County using Kernel Density Estimation (KDE)

Kernel Density Estimation (KDE) is a statistical technique for estimating the magnitude of an event per area unit (e.g. reports per square km)

Background

Complaint locations were filtered by calendar year and clipped to the Collier County study boundary. For each year, points were projected into NAD83 / UTM Zone 17N so distances were measured in meters.

A kernel density estimate was calculated for each year using a 1,500-meter smoothing bandwidth. The resulting surface was converted to reports per square kilometer, clipped to the county boundary, and displayed with a fixed 0 to 18 reports per km² color scale across all yearly maps.

  • Darker/brighter KDE areas indicate stronger complaint clustering.
  • White contours show high-density zones based on KDE percentile breaks.
  • The KDE should be interpreted as a smoothed complaint-intensity surface, not a direct mosquito abundance estimate using trap data
Data source: Collier Mosquito GIS service request layer and Collier County boundary layer. Generated using R, sf, spatstat, raster, leaflet, and htmlwidgets.
